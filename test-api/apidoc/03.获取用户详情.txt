获取用户详情
接口地址：/getUserDetail
存在登录校验：是
请求类型：GET
请求参数：
    id:用户ID
成功状态示例
    {
        code:200,
        data:{
            "id": 1,               // 用户ID
            "username": "imooc",   // 用户名
            "password": "123456"   // 密码
        }
    }

失败状态示例
    {
        code:400,
        message: "未登录"
    }
