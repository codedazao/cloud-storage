获取用户列表
接口地址：/getUserList
存在登录校验：是
请求类型：GET
请求参数：
    无
成功状态示例
    {
        code:200,
        data:[
           {
                "id": 1,               // 用户ID
                "username": "imooc",   // 用户名
                "password": "123456"   // 密码
           },
           {
                "id": 2,
                "username": "imooc2",
                "password": "123456"
           },
           ......
         ]
    }
失败状态示例
    {
        code:400,
        message: "未登录" // 失败原因
    }
