全局说明

1、接口前缀：http://127.0.0.1:3000

2、数据提交格式：除特殊情况如（文件上传），其他统一为 JSON 格式提交数据
3、数据返回给是：除特殊情况如（文件数据），其他统一为 JSON 格式返回数据

4、返回数据格式
    code => 数据状态
    data => 数据内容（code 为 200 时存在）
    message => 数据消息 （code 为 300 时存在）

5、全局code说明
    200 => 普通成功状态
    300 => 普通错误状态
    301 => 请求参数错误
    400 => 未登录
    401 => 登录状态失效

6、登录状态校验规则
    需在请求头中携带如下参数，以便进行登录校验
    request.header:
        x-id => 用户ID
        x-time => 登录时间
        x-token => 登录令牌

7、登录有效时间 86400秒（1天）
